# Other Trails Directory

This directory contains additional GPX trail files beyond the official Skåneleden trails.

## Purpose

- Store regional trails in Skåne
- Supplementary hiking routes
- Local favorites and custom routes

## Usage

1. Drop GPX files into this directory
2. Files are automatically loaded by the app
3. Toggle "Use World Wide Hikes?" to OFF to view these trails

## Data Privacy

This directory is excluded from version control (see `.gitignore`).
Your personal trail data remains private.
