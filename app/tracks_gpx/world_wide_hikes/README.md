# World Wide Hikes Directory

This directory contains GPX files for hikes from around the world.

## Purpose

- Store international hiking trails
- Travel hiking memories
- Routes from outside Skåne/Sweden

## Usage

1. Upload GPX files through the app's file uploader
2. Or manually drop GPX files into this directory
3. Toggle "Use World Wide Hikes?" to ON to view these trails

## Data Privacy

This directory is excluded from version control (see `.gitignore`).
Your personal trail data remains private.
